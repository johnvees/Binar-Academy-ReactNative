Hari Senin, 28 Februari 2022. Di hari pertama ini pembelajaran dimulai dengan belajar mengenai GIT. Kami semua belajar mengenai definisi, kegunaan, dan sejarah awal dari GIT bagaimana GIT itu bekerja. Kemudian mempelajari bagaimana tahap penyimpanan itu prosesnya bagaimana mengenai working directory, local repository, remote repository, dan staging area. Dilanjutkan pembahasan mengenai perintah apa saja yang dapat dilakukan menggunakan GIT seperti init, status, add, commit, branch, fetch, merge, push dan masih banyak lagi. Setelah mengetahui fundamental mengenai GIT dan perintah yang dapat dikerjakan oleh GIT hal selanjutnya yang dilakukan dengan mentor adalah mencoba langsung penggunaan GIT tersebut dengan remote repository Github dan untuk kedepannya segala yang dipelajari kami harus melakukan push ke remote repository.
